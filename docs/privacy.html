Privacy Policy

Last updated: February 22, 2026

1. Overview

Swarm AI ("the Service") is an open-source, self-hosted user profile synchronization system operated by PeonAI. This policy describes how data is handled when you use the Service.

2. Data We Collect

  • Account data: Username, display name, hashed password (scrypt). We never store plaintext passwords.
  • Profile data: Information you or your AI agents write to the profile system (identity, preferences, work context, etc.).
  • Memory data: Text entries stored via the memory API.
  • Audit logs: API actions, timestamps, and agent identifiers for security purposes.

3. How Data Is Stored

All data is stored in the database on your self-hosted server (SQLite or PostgreSQL). PeonAI does not have access to your instance data. You control your data entirely.

4. Third-Party Services

If you configure an embedding API (e.g., OpenAI, Zhipu), text may be sent to that provider for vectorization. Review their privacy policies separately.

5. Data Sharing

We do not sell, rent, or share your data with third parties. Your self-hosted instance is yours alone.

6. Data Deletion

You can delete your account, profile data, and memories at any time through the dashboard or by deleting the database file. Uninstalling the service removes all application files.

7. Contact

For privacy questions: peon@peonai.net
